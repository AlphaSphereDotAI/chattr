## [0.0.100] - 2024-01-28

### Added
- Initial release of Chattr
- Multi-modal AI chat with historical characters
- Gradio-based web interface
- Agno agent framework integration
- Qdrant vector database for knowledge storage
- MCP integration for audio and video generation
- Docker and Docker Compose deployment support
- PII detection and prompt injection guardrails
- Memory and context management
- Character prompt templating with POML
- Support for Napoleon and Einstein characters

### Features
- Text-based conversations
- Audio generation from responses
- Video synthesis with character animations
- Persistent conversation history
- Semantic knowledge search
- Streaming response support
- Health checks and monitoring
- PWA support
- Auto-generated REST API

### Technical
- Python 3.13+ support
- Type hints throughout codebase
- Pydantic-based configuration
- Comprehensive test suite
- CI/CD with GitHub Actions
- Docker multi-stage builds
- Security scanning with CodeQL
